Html is the skeleton of the web page or we can say it as as structure of page.

CSS is used for designing of our html pages.


In the Html there are Three Types of Styling

  1. Inline Styling --- If we want to design using inline styling we want mention style attribute.
  2. Internal Styling --- If we want to design Internal Styling We have to use style tag in our head tag and then our element name or class or ids in our head tag.
  3. External Styling --- If we want to design using External Styling we need to create CSS file and link it with the HTML file using link tag and then we have to give our element name or class or ids in our head tag.
  • Inline Styling has first priority
  • Internal Styling has 2nd priority
  • External Stylung is the last priority
